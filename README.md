<!-- GETTING STARTED -->

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/chelseaharris137/Quiz-App.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app in the development mode.
   ```JS
    npm run dev
   ```
4. Open http://localhost:8080 to view it in the browser.
